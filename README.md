Movie Share
======================

Movie Share is a hybrid app developed in Phonegap + AngularJS + Ionic + Firebase, for finding movie information and keeping track of the movies you want to see and those you have already seen.

## Getting Started

**Install Ionic**

If you haven't installed Ionic already, follow the instructions [here](http://ionicframework.com/getting-started/).

**Emulate the app in a Browser**

    $ phonegap serve
    $ http://localhost:3000
